
Sitio multipágina - Franklin Delano Roosevelt
--------------------------------------------

Archivos incluidos en la carpeta 'fdr_multipage_site':
- index.html
- carreras.html
- carrera_*.html (una por cada carrera)
- noticias.html, eventos.html, anuncios.html, campus.html, promo2025.html, contacto.html
- evento_detalle.html, anuncio_detalle.html (páginas de ejemplo para noticias/eventos)
- assets/css/style.css
- images/ (carpeta vacía para tus fotos)

Cómo usar y mostrar a tu profesor:
1) Descarga el zip y descomprímelo.
2) Abre 'index.html' con doble clic en cualquier navegador o usa Live Server en VS Code para preview con recarga automática.
3) Para reemplazar placeholders por imágenes reales: coloca las fotos en la carpeta 'images/' y en los archivos HTML cambia el bloque <div class="img-pl">... por: <img src="images/mi-foto.jpg" alt="..." style="width:100%;height:auto;border-radius:8px">

Notas:
- La navegación ya abre páginas independientes (sin JavaScript) como pediste: cada carrera tiene su propia página.
- El título en la página de inicio tiene una animación CSS por palabra. Si quieres otra animación (efecto máquina de escribir, aparición por letra, etc.) dime cuál prefieres.
- Si quieres que te entregue este contenido en un ZIP listo para descargar, lo genero ahora.
